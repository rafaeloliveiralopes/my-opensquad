# Criar copy do feed

Escreva um carrossel ou post estático de Instagram Feed com foco em autoridade e conversão consultiva.

## Process

1. Ler o ângulo escolhido e a direção do CTA.
2. Escrever hook, desenvolvimento e CTA com linguagem simples e estratégica.
3. Adaptar a peça para carrossel ou estático, conforme definido no brief.
4. Entregar legenda pronta junto com o corpo principal do post.

## Output Format

```markdown
# Formato
carrossel | estatico

# Hook
...

# Corpo
Slide 1: ...

# Legenda
...

# CTA
...
```

## Quality Criteria

- [ ] hook forte
- [ ] dor clara para clínicas
- [ ] CTA consultivo e contextual
