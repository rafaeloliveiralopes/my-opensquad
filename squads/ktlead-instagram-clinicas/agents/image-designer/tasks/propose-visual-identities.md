# Propor identidades visuais

Crie 3 direções visuais para o post, alinhadas ao posicionamento premium e estratégico da KTLead.

## Process

1. Ler o copy e identificar emoção dominante.
2. Propor 3 identidades com diferença real entre si.
3. Documentar paleta, tipografia e sensação visual.

## Output Format

```yaml
visuals:
  - id: 1
    name: "..."
    mood: "..."
    palette: "..."
    typography: "..."
```

## Quality Criteria

- [ ] 3 opções bem distintas
- [ ] hierarquia visual clara
- [ ] coerência com clínicas e autoridade
