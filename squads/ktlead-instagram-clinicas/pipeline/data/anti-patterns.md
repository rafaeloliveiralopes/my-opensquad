# Anti-Patterns

- Falar de marketing de forma genérica sem conectar com clínica
- Fazer post bonito sem tese forte
- Usar CTA vazio como "me chama" sem contexto
- Falar só de likes, alcance e vaidade
- Produzir conteúdo técnico demais para público local
- Parecer uma agência de post, não uma parceira de crescimento
- Exagerar promessas sem prova ou lógica
